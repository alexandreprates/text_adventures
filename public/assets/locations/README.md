# Location Art

## Village Hub

`village-hub.png` is an original AI-generated pixel-art town hub created for Text Adventures.

The image represents the town of Nee'Peh as a compact fantasy village containing the tavern, blacksmith, armorsmith, priest temple, merchant stalls, and ruins/dungeon entrance. It is designed as a visual replacement for the purely textual town map in the web frontend.

Recommended future use:

- Add invisible hotspots for town destinations.
- Keep command buttons and text input as the primary interaction layer.
- Preserve the text fallback for accessibility and debug use.

## Source

- Generated specifically for this repository using the Codex image generation workflow.
- Prompt requested an original pixel-art fantasy village in the same style direction as the dungeon tileset concept.
- No external RPG Maker asset was copied or adapted directly.
