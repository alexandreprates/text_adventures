# Location Art

## Village Hub

`village-hub.png` is an original AI-generated pixel-art town hub created for Text Adventures.

The image represents the town of Nee'Peh as a compact fantasy village containing the tavern, blacksmith, armorsmith, priest temple, merchant stalls, and ruins/dungeon entrance. It is designed as a visual replacement for the purely textual town map in the web frontend.

## Tavern Interior

`tavern-interior.png` is an original AI-generated pixel-art tavern scene created for Text Adventures.

The image represents the tavern as a warm rest and supply location, with a fireplace, bar counter, tables, bottles, and rented-room access.

## Merchant District

`merchant-district.png` is an original AI-generated pixel-art merchant/workshop scene created for Text Adventures.

The image represents the blacksmith and armorsmith scenes with a forge, weapon racks, armor displays, counters, crates, and trading stalls.

## Temple Sanctuary

`temple-sanctuary.png` is an original AI-generated pixel-art temple scene created for Text Adventures.

The image represents Aluriel's Priest as a healing and magic location, with altar, fountain, candles, stained glass, tome shelves, and ritual space.
