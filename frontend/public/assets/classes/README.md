# Class Sprites

`class-spritesheet.png` contains original transparent chibi pixel-art player class sprites.

- Cell size: 32x32 px
- Layout: 4 columns x 4 rows
- Order: Adventurer, Blademaster, Dragoon, Nightblade, Arcanist, Druid, Warlord, Duelist, Spellblade, Warden, Skirmisher, Battlemage, Sentinel, Hexblade, Ranger, Mystic
- Individual sprite exports are kept in `sprites/` for inspection and future iteration.
